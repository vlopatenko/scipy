# Заранее определим функцию, которая будет принимать на вход n и возвращать выборочные средние для выборок объема n
def samples_average(n):
    samples = np.array([sts.expon().rvs(n) for _ in range(N)])
    # Считаем средние для каждой выборки
    return np.mean(samples, axis=1)

Выводы по результатам¶

В качестве исходного распределения мы взяли экспоненциальное, которое сильно скошено по сравнению с нормальным и совсем на него непохоже. Как видно, для выборок объема 5 распределение все еще не очень походит на нормальное, однако с ростом n оно постепенно начинает приобретать форму шляпы, становится менее скошенным, а гистограмма по форме все ближе приближается к теоретической плотности соответствующего нормального распределения, так, точность аппроксимации с ростом n заметно растет (особенно это начинает прослеживаться, начиная со значения n = 50). Таким образом, центральная предельная теорема работает! (даже для такого скошенного распределения)
